# pacman-ai-python
Pac-Man game implemented in Python using Pygame with User Mode and Utility-Based AI Mode


      -------Pac-Man AI Project (Python & Pygame)----------
This project is a Pac-Man game implementation developed using Python and Pygame, featuring both User-Controlled gameplay and an AI-controlled Pac-Man powered by a Utility-Based Decision Making Algorithm.

The project was developed as part of artificial intelligence and game programming practice, focusing on decision-making under uncertainty, heuristic evaluation, and real-time game simulation.
